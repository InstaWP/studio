How it works

Three things, that's the whole model.

1. Author

An AI agent (or you) writes an HTML file into site/. Drop a file, get a page.

2. Render

The theme reads the file live and serves it as a real WordPress page. No import, no drift.

3. Review & ship

Collect pin feedback, resolve it with an agent, and push to InstaWP when it's ready.
